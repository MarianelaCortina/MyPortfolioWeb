    Sistema de Ventas

    Realicé el Sistema de Ventas utilizando Angular 16 y para el backend .Net Core 8 con MS SQL Server.
    Este sistema permite manejar diferentes tipos de usuarios (Administrador, Supervisor, Empleado), quienes tendrán diferentes tipos de acceso a las funcionalidades, según su rol.
    Cuenta con un Dashboard en el que se puede visualizar un gráfico de barras con las ventas realizadas en la última semana. También con secciones: Registrar venta, Productos, Historial de Ventas y Reportes.

    La app del clima

    WeatherApp la realicé con Angular 13. Cuando ingreses a la app, te pedirá permiso para conocer tu ubicación, así te podrá mostrar el clima de tu ciudad en tiempo real. Esto está logrado gracias al uso de geo-localicación, a través del uso de la API OpenWeatherMap (https://api.openweathermap.org). Cuando le das permiso, toma las coordenadas de la ciudad donde te encuentras, se las envía a la API y esta te devuelve la respuesta cargando la temperatura en la app. También podrás buscar la temperatura de la ciudad que escribas en el buscador.
