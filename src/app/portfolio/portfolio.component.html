    Sistema de Ventas

    Realicé el Sistema de Ventas utilizando Angular y para el backend .Net Core con MS SQL Server.

    ❇️Autenticación y seguridad: Implementé un sistema de login con autenticación basada en tokens JWT, garantizando la seguridad y protección de las sesiones de los usuarios.
    ❇️Cuenta con un Dashboard en el que se puede visualizar un gráfico de barras con las ventas realizadas en la última semana.

    La app del clima

    WeatherApp la realicé con utilizando Angular versión 13.
    *️⃣Cuando ingreses a la app, te pedirá permiso para conocer tu ubicación, así te podrá mostrar el clima de tu ciudad en tiempo real.
    *️⃣Esto está logrado gracias al uso de geo-localicación, a través del uso de la API OpenWeatherMap.
    *️⃣Cuando le das permiso, toma las coordenadas de la ciudad donde te encuentras, se las envía a la API y esta te devuelve la respuesta cargando la temperatura en la app.
    *️⃣También podrás buscar la temperatura de la ciudad que escribas en el buscador.
